Pool settings
Solo payout address
This is the address Miningcore uses when generating blocks. Restart the app after saving.
Use a DigiByte legacy/Base58 address (typically starting with D or S). AxeDGB validates the address locally using your node RPC. If the node is still starting, saving may fail; try again in a few minutes.
Difficulty (vardiff)
These presets tune the SHA256 pool share difficulty for DigiByte's fast blocks. Save + restart to apply. (Scrypt uses its own defaults.)
Pick a preset based on each miner's hashrate. VarDiff is set per miner connection, not your total fleet hashrate. Setting difficulty too high won't help you find blocks. It just reduces how often shares are submitted and can waste hashing effort.
Changing difficulty requires an app restart.
Lowest difficulty Miningcore will assign.
Initial difficulty for new miners.
0 means no limit.
After saving: restart the app from 5tratumOS to apply changes.
